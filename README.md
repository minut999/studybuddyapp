# Studybuddy 

StudyBuddy is a platform designed for university students to track and plan their studies.

# features
- [x] Pomodoro timer 
- [x] Todo list
- [x] Calendar
- [x] Notes
- [x] deadlines
# tech stack 
- Vue 3
- Vite 
- pnpm


# setup

install

```bash
git clone https://github.com/studybuddydev/studybuddy-app
pnpm i
```
run 
    
```bash
pnpm dev
```
